# Developer Portfolio Hero

A premium, interactive personal portfolio website featuring a seamless dark-theme hero section, live typography customization, responsive layouts, and modern visual effects.

## Features

- **Liquid Glass Header:** Floating navigation pill with blur effects.
- **Visual Portfolio Hero:** Bold headings with active typography configuration.
- **Projects & Skills Sections:** Interactive showcase of technical abilities and works.
- **Live Customizer Panel:** Instant configuration drawer for portfolio bio, skills, CTA buttons, and social connections.

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the application:
   ```bash
   npm run dev
   ```
